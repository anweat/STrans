## 一、算法总览

```
输入帧 (BGR ndarray)
    │
    ├─① 车辆检测 (YOLOv8-nano)
    │     输出: [{bbox, class, confidence}]
    │
    ├─② 多目标跟踪 (ByteTrack)
    │     输入: ①的检测结果
    │     输出: [{track_id, bbox, class, confidence}]
    │
    ├─③ 车流量统计 (ROI 跨线计数)
    │     输入: ②的 track_id + 轨迹
    │     输出: count_in, count_out, current_count
    │
    ├─④ 车速估计 (像素位移法 + 标定)
    │     输入: ②的 track_id 连续帧位置
    │     输出: 每辆车 speed_kmh
    │
    ├─⑤ 车牌识别 (HyperLPR3)
    │     输入: ①检测到的车辆区域裁剪图
    │     输出: plate_text 字符串
    │
    ├─⑥ 拥堵检测 (密度 + 速度双阈值)
    │     输入: ③的密度 + ④的平均速度
    │     输出: congestion_level (低/中/高)
    │
    ├─⑦ 事故/异常检测 (异常运动规则)
    │     输入: ②的轨迹 + ④的速度
    │     输出: event {type, description, severity}
    │
    ├─⑧ 红绿灯识别 (HSV 颜色阈值 + 多帧投票)
    │     输入: 原始帧
    │     输出: {state: red/green/yellow, confidence}
    │
    └─⑨ 障碍物识别 (YOLOv8-nano 扩展类别)
          输入: 与①同一帧（复用检测结果，零额外算力）
          输出: [{bbox, class: "person"/"cone"/"barrier", confidence}]

    └─⑩ 车流密度热力图 (accumulateWeighted + JET 伪彩映射)
          输入: ②的 track_id + bbox 中心点（每帧）
          输出: 热力图 base64 + hotspots 热点列表
```

**依赖关系：** ① → ② → ③④⑤⑩；⑥ 依赖 ③④；⑦ 依赖 ②④；⑧ 独立运行；⑨ 与①共享检测结果；⑩ 依赖 ②

## 二、逐项算法详解

### ① 车辆检测 — YOLOv8-nano

| 项 | 内容 |
|---|---|
| **模型** | YOLOv8n（nano 版本，权重 6.3MB） |
| **框架** | ultralytics（pip install ultralytics） |
| **输入** | 640×640 BGR 图像（自动 letterbox 缩放） |
| **输出** | `[{bbox: [x1,y1,x2,y2], class: "car", confidence: 0.92}]` |
| **检测类别** | car（轿车）、truck（卡车）、bus（公交）、motorcycle（摩托）|
| **CPU 帧率** | ~15-20 FPS（笔记本 i5/i7） |
| **GPU 帧率** | ~60+ FPS（RTX 4060） |

**核心代码逻辑：**

```python
from ultralytics import YOLO

class VehicleDetector:
    def __init__(self, model_path="yolov8n.pt", conf=0.5, iou=0.45):
        self.model = YOLO(model_path)
        self.conf = conf
        self.iou = iou
        # COCO 类别中 2=car, 5=bus, 7=truck, 3=motorcycle
        self.vehicle_classes = [2, 3, 5, 7]
        self.class_names = {2: "car", 3: "motorcycle", 5: "bus", 7: "truck"}

    def detect(self, frame):
        """
        输入: BGR ndarray (H, W, 3)
        输出: list of {bbox, class, confidence}
        """
        results = self.model(frame, conf=self.conf, iou=self.iou, verbose=False)
        detections = []
        for box in results[0].boxes:
            cls_id = int(box.cls[0])
            if cls_id in self.vehicle_classes:
                detections.append({
                    "bbox": [int(x) for x in box.xyxy[0].tolist()],
                    "class": self.class_names[cls_id],
                    "confidence": float(box.conf[0]),
                })
        return detections
```

**为什么选 YOLOv8n 而不是别的：**

| 对比项 | YOLOv8n | YOLOv8s | Faster R-CNN | SSD512 |
|--------|---------|---------|--------------|--------|
| 权重大小 | 6.3MB | 22MB | 58MB | 100MB+ |
| CPU FPS | 15-20 | 8-10 | 2-3 | 5-8 |
| mAP(COCO) | 37.3 | 44.9 | 42.0 | 28.8 |
| 部署难度 | 低 | 低 | 高 | 中 |
| **结论** | **采用** | 精度更高但慢 | 太慢不适合实时 | 精度不够 |

### ② 多目标跟踪 — ByteTrack

| 项 | 内容 |
|---|---|
| **算法** | ByteTrack（不依赖外观特征的跟踪算法） |
| **安装** | `pip install bytetracker` 或直接用 ultralytics 内置 `model.track()` |
| **输入** | ①的检测结果列表 |
| **输出** | `[{track_id: 7, bbox: [x1,y1,x2,y2], class: "car", confidence: 0.92}]` |
| **原理** | 用检测框的 IoU 关联前后帧，低分检测也参与二次匹配 |
| **优势** | 不需要 ReID 模型（省算力），遮挡恢复快 |

**核心代码逻辑：**

```python
class VehicleTracker:
    def __init__(self):
        # 方案A: 直接用 ultralytics 内置跟踪（最简单）
        self.model = YOLO("yolov8n.pt")
        self.vehicle_classes = [2, 3, 5, 7]
        self.class_names = {2: "car", 3: "motorcycle", 5: "bus", 7: "truck"}

    def track(self, frame):
        """
        输入: BGR ndarray
        输出: list of {track_id, bbox, class, confidence}
        """
        # ultralytics 内置 ByteTrack
        results = self.model.track(
            frame, persist=True, tracker="bytetrack.yaml",
            conf=0.5, iou=0.45, verbose=False
        )
        tracked = []
        for box in results[0].boxes:
            cls_id = int(box.cls[0])
            if cls_id in self.vehicle_classes and box.id is not None:
                tracked.append({
                    "track_id": int(box.id[0]),
                    "bbox": [int(x) for x in box.xyxy[0].tolist()],
                    "class": self.class_names[cls_id],
                    "confidence": float(box.conf[0]),
                })
        return tracked
```

**为什么选 ByteTrack 而不是 DeepSORT：**

| 对比项 | ByteTrack | DeepSORT |
|--------|-----------|----------|
| 是否需要 ReID 模型 | 不需要 | 需要（额外 30MB+） |
| CPU 跟踪耗时 | ~2ms/帧 | ~15ms/帧 |
| ID 切换率 | 低 | 中 |
| 遮挡恢复 | 好 | 一般 |
| 部署复杂度 | 低 | 中 |
| **结论** | **采用** | 算力够时可选 |

### ③ 车流量统计 — ROI 跨线计数

| 项 | 内容 |
|---|---|
| **算法** | 虚拟计数线 + 跟踪轨迹跨线判定 |
| **输入** | ②的 track_id + bbox 中心点轨迹历史 |
| **输出** | `count_in`（进入数）、`count_out`（离开数）、`current_count`（画面内当前数） |
| **原理** | 在画面中画一条虚拟线，记录每个 track_id 的中心点轨迹，当中心点从线的一侧变到另一侧时计数 +1 |

**核心代码逻辑：**

```python
import numpy as np

class FlowCounter:
    def __init__(self, line_start=(0, 400), line_end=(1280, 400)):
        """
        line_start, line_end: 计数线的两个端点（像素坐标）
        默认在画面中间画一条水平线
        """
        self.line_start = np.array(line_start)
        self.line_end = np.array(line_end)
        self.count_in = 0
        self.count_out = 0
        self.tracked_centers = {}  # {track_id: previous_center}

    def _cross_line(self, prev_center, curr_center):
        """
        判断中心点是否跨过计数线
        用向量叉积判断点在线的哪一侧
        返回: 1=从上到下(in), -1=从下到上(out), 0=未跨线
        """
        line_vec = self.line_end - self.line_start
        prev_side = np.cross(line_vec, prev_center - self.line_start)
        curr_side = np.cross(line_vec, curr_center - self.line_start)

        if prev_side * curr_side < 0:  # 符号相反 = 跨线
            return 1 if curr_side > 0 else -1
        return 0

    def update(self, tracked_objects):
        """
        输入: [{track_id, bbox, ...}]
        输出: {count_in, count_out, current_count}
        """
        current_ids = set()
        for obj in tracked_objects:
            tid = obj["track_id"]
            bbox = obj["bbox"]
            curr_center = np.array([(bbox[0] + bbox[2]) / 2, (bbox[1] + bbox[3]) / 2])
            current_ids.add(tid)

            if tid in self.tracked_centers:
                prev_center = self.tracked_centers[tid]
                direction = self._cross_line(prev_center, curr_center)
                if direction > 0:
                    self.count_in += 1
                elif direction < 0:
                    self.count_out += 1

            self.tracked_centers[tid] = curr_center

        # 清理已离开的 track_id
        left_ids = set(self.tracked_centers.keys()) - current_ids
        for lid in left_ids:
            del self.tracked_centers[lid]

        return {
            "count_in": self.count_in,
            "count_out": self.count_out,
            "current_count": len(current_ids),
        }
```

**计数线配置说明：**

```
画面坐标示意（以 1280×720 为例）：

  (0,0) ─────────────────────────────── (1280,0)
    │                                       │
    │           车辆行驶方向 ↓               │
    │                                       │
    │ ────── 计数线 (0,360)→(1280,360) ──── │  ← 在画面中间画线
    │                                       │
    │                                       │
  (0,720) ─────────────────────────────── (1280,720)

  车从上往下经过线 → count_in + 1
  车从下往上经过线 → count_out + 1
```

### ④ 车速估计 — 像素位移标定法

| 项 | 内容 |
|---|---|
| **算法** | 像素位移 / 帧间时间 × 像素→米比例系数 |
| **输入** | ②的 track_id 在连续帧中的中心点位置 |
| **输出** | 每辆车 `speed_kmh` |
| **标定方法** | 在沙盘上量一段已知距离 → 在画面中标出两端像素坐标 → 计算像素/米比例 |

**标定公式：**

```
pixels_per_meter = 像素距离 / 实际距离(米)

单帧位移(米) = 像素位移 / pixels_per_meter

瞬时速度(m/s) = 单帧位移(米) × FPS

车速(km/h) = 瞬时速度(m/s) × 3.6
```

**核心代码逻辑：**

```python
import collections

class SpeedEstimator:
    def __init__(self, pixels_per_meter=50.0, fps=15, smooth_window=5):
        """
        pixels_per_meter: 标定系数（需实测）
        fps: 视频帧率
        smooth_window: 滑动平均窗口大小（帧）
        """
        self.ppm = pixels_per_meter
        self.fps = fps
        self.smooth_window = smooth_window
        self.position_history = {}  # {track_id: deque([center1, center2, ...])}

    def estimate(self, tracked_objects):
        """
        输入: [{track_id, bbox, ...}]
        输出: {track_id: speed_kmh}
        """
        speeds = {}
        for obj in tracked_objects:
            tid = obj["track_id"]
            bbox = obj["bbox"]
            center = ((bbox[0] + bbox[2]) / 2, (bbox[1] + bbox[3]) / 2)

            if tid not in self.position_history:
                self.position_history[tid] = collections.deque(maxlen=self.smooth_window)

            history = self.position_history[tid]
            history.append(center)

            if len(history) >= 2:
                # 计算最近 N 帧的总位移
                total_pixel_dist = 0
                for i in range(1, len(history)):
                    dx = history[i][0] - history[i-1][0]
                    dy = history[i][1] - history[i-1][1]
                    total_pixel_dist += (dx**2 + dy**2) ** 0.5

                # 平均单帧位移
                avg_pixel_per_frame = total_pixel_dist / (len(history) - 1)
                # 转换为 m/s
                speed_ms = avg_pixel_per_frame / self.ppm * self.fps
                # 转换为 km/h
                speed_kmh = speed_ms * 3.6
                speeds[tid] = round(speed_kmh, 1)

        return speeds
```

**标定实操步骤：**

| 步骤 | 操作 | 示例 |
|------|------|------|
| 1 | 在沙盘上量一段距离 | 量出 1 米 |
| 2 | 在画面中找到这段距离的两端 | A 点像素 (100, 300)，B 点像素 (350, 300) |
| 3 | 计算像素距离 | √((350-100)² + 0²) = 250 像素 |
| 4 | 计算比例系数 | 250 像素 / 1 米 = 250 pixels_per_meter |
| 5 | 写入代码 | `SpeedEstimator(pixels_per_meter=250)` |

### ⑤ 车牌识别 — HyperLPR3

| 项 | 内容 |
|---|---|
| **模型** | HyperLPR3（车牌检测 + 四点透视矫正 + CRNN 字符识别） |
| **框架** | hyperlpr3（pip install hyperlpr3） |
| **输入** | ①检测到的车辆区域裁剪图 |
| **输出** | 车牌字符串 `plate`（如 "京A12345"） |
| **精度** | CCPD 数据集 95%+ 识别准确率 |
| **速度** | CPU ~5-10ms/张 |
| **模型大小** | ~10MB（轻量，无需安装 PaddlePaddle 全家桶） |
| **车牌类型** | 大陆蓝牌/绿牌(新能源)/黄牌/白牌/警牌，港澳车牌 |

**核心代码逻辑：**

```python
from hyperlpr3 import LicensePlateCatcher

class PlateRecognizer:
    def __init__(self):
        # HyperLPR3 一行初始化，自带检测+矫正+识别
        self.catcher = LicensePlateCatcher()

    def recognize(self, vehicle_img):
        """
        输入: 车辆区域裁剪图 (BGR ndarray)
        输出: 车牌字符串 或 None
        """
        # HyperLPR 直接返回 [车牌坐标, 车牌文本, 置信度] 列表
        results = self.catcher(vehicle_img)
        if not results:
            return None

        # 取置信度最高的结果
        best = max(results, key=lambda r: r[1])
        plate_text = best[0]
        confidence = best[1]

        if confidence > 0.7:
            return plate_text
        return None
```

| **结论** | **采用** | 备选（更重） | 不推荐 |

### ⑥ 拥堵检测 — 密度 + 速度双阈值

| 项 | 内容 |
|---|---|
| **算法** | 车辆密度 + 平均速度双指标阈值判定 |
| **输入** | ③的 `current_count`（当前车辆数）、画面面积、④的平均速度 |
| **输出** | `congestion_level`：低 / 中 / 高 |
| **时间窗口** | 5 秒滑动窗口平均值（避免单帧抖动） |

**计算公式：**

```
车辆密度 = current_count / 画面面积(千像素²)
    例: 12 辆 / (1280×720/1000) = 12 / 921.6 = 0.013 辆/千像素²

平均速度 = 所有 tracked 车辆 speed_kmh 的均值
```

**判定规则：**

| 拥堵等级 | 车辆密度 | 平均车速 | 含义 |
|----------|----------------|---------|------|
| **低（畅通）** | < 0.05 | > 30 km/h | 车少速度快 |
| **中（缓行）** | 0.05 ~ 0.10 | 15 ~ 30 km/h | 车多速度中 |
| **高（拥堵）** | > 0.10 | < 15 km/h | 车多速度慢 |


**核心代码逻辑：**

```python
import collections
import time

class CongestionDetector:
    def __init__(self, frame_area=921.6, window_seconds=5, fps=15):
        """
        frame_area: 画面面积（千像素²），1280×720 → 921.6
        window_seconds: 滑动窗口秒数
        """
        self.frame_area = frame_area
        self.window_size = window_seconds * fps  # 窗口内帧数
        self.density_history = collections.deque(maxlen=self.window_size)
        self.speed_history = collections.deque(maxlen=self.window_size)

        # 阈值（可前端配置）
        self.density_low = 0.05
        self.density_high = 0.10
        self.speed_low = 15.0
        self.speed_high = 30.0

    def detect(self, current_count, speeds_dict):
        """
        输入:
            current_count: 当前画面车辆数
            speeds_dict: {track_id: speed_kmh}
        输出: congestion_level ("低"/"中"/"高")
        """
        density = current_count / self.frame_area
        avg_speed = sum(speeds_dict.values()) / len(speeds_dict) if speeds_dict else 0

        self.density_history.append(density)
        self.speed_history.append(avg_speed)

        # 滑动窗口平均
        avg_density = sum(self.density_history) / len(self.density_history)
        avg_speed_smoothed = sum(self.speed_history) / len(self.speed_history)

        # 密度判定
        if avg_density < self.density_low:
            density_level = 0  # 低
        elif avg_density < self.density_high:
            density_level = 1  # 中
        else:
            density_level = 2  # 高

        # 速度判定
        if avg_speed_smoothed > self.speed_high:
            speed_level = 0  # 低
        elif avg_speed_smoothed > self.speed_low:
            speed_level = 1  # 中
        else:
            speed_level = 2  # 高

        # 取更严重的等级
        level = max(density_level, speed_level)
        return ["低", "中", "高"][level]
```

### ⑦ 事故/异常检测 — 规则法

| 项 | 内容 |
|---|---|
| **算法** | 基于轨迹和速度的规则判定（无需训练） |
| **输入** | ②的 track_id 轨迹历史 + ④的速度 |
| **输出** | `event {type, description, severity, bbox}` |
| **检测类型** | 异常停车、逆行 |

**规则 1：异常停车检测**

```
触发条件：
  - 某 track_id 的速度 < 2 km/h
  - 持续时间 > 3 秒（45 帧 @15FPS）
  - 且该 track_id 在画面内（未离开）

输出事件:
  {type: "accident", description: "车辆异常停车", severity: "warning"}
```

**规则 2：逆行检测**

```
触发条件：
  - 预设正常行驶方向（如从上到下 = y 增大）
  - 某 track_id 在 1 秒内 y 方向位移 < -50 像素（从下往上走）
  - 即与预设方向相反

输出事件:
  {type: "illegal", description: "车辆逆行", severity: "critical"}
```

**核心代码逻辑：**

```python
import collections
import time

class AccidentDetector:
    def __init__(self, fps=15, stop_threshold=2.0, stop_duration=3.0):
        """
        stop_threshold: 停车速度阈值 km/h
        stop_duration: 持续多少秒判定为异常停车
        """
        self.fps = fps
        self.stop_threshold = stop_threshold
        self.stop_frames = int(stop_duration * fps)
        self.speed_history = {}  # {track_id: deque([speed1, speed2, ...])}
        self.alerted = set()  # 已告警的 track_id，避免重复

    def detect(self, tracked_objects, speeds_dict):
        """
        输入:
            tracked_objects: [{track_id, bbox, ...}]
            speeds_dict: {track_id: speed_kmh}
        输出: list of events
        """
        events = []
        current_ids = set()

        for obj in tracked_objects:
            tid = obj["track_id"]
            speed = speeds_dict.get(tid, 0)
            current_ids.add(tid)

            if tid not in self.speed_history:
                self.speed_history[tid] = collections.deque(maxlen=self.stop_frames)

            self.speed_history[tid].append(speed)

            # 检查是否持续低速
            history = self.speed_history[tid]
            if len(history) >= self.stop_frames:
                if all(s < self.stop_threshold for s in history):
                    if tid not in self.alerted:
                        events.append({
                            "type": "accident",
                            "description": f"车辆#{tid}异常停车",
                            "severity": "warning",
                            "bbox": obj["bbox"],
                        })
                        self.alerted.add(tid)

        # 清理已离开的
        left = set(self.speed_history.keys()) - current_ids
        for lid in left:
            del self.speed_history[lid]
            self.alerted.discard(tid)

        return events
```

### ⑧ 红绿灯识别 — HSV 颜色阈值 + 多帧投票

| 项 | 内容 |
|---|---|
| **算法** | HSV 颜色空间阈值分割 + 形态学滤波 + 多帧投票 |
| **输入** | 原始帧 |
| **输出** | `{state: "red"/"green"/"yellow", confidence: 0.95}` |
| **优势** | 无需训练，计算量极小（<1ms/帧） |

**HSV 阈值表：**

| 颜色 | H 范围 | S 范围 | V 范围 | 说明 |
|------|--------|--------|--------|------|
| **红色** | 0~10 或 170~180 | 100~255 | 100~255 | 红色在 HSV 两端，需两个区间 |
| **绿色** | 35~85 | 100~255 | 100~255 | |
| **黄色** | 20~35 | 100~255 | 100~255 | |

**处理流程：**

```
原始帧 (BGR)
    │
    ▼
BGR → HSV 转换 (cv2.cvtColor)
    │
    ▼
对红/绿/黄三个颜色分别做 inRange 阈值分割
    │
    ▼
对每个颜色掩码做形态学开运算（去噪点）
    │
    ▼
找轮廓 → 过滤面积太小的 → 计算各颜色像素总面积
    │
    ▼
取面积最大的颜色作为当前帧判定
    │
    ▼
最近 5 帧投票 → 多数票为最终结果
```

**核心代码逻辑：**

```python
import cv2
import numpy as np
import collections

class TrafficLightDetector:
    def __init__(self, vote_window=5, min_area=50):
        """
        vote_window: 投票窗口帧数
        min_area: 颜色区域最小像素面积（过滤噪点）
        """
        self.vote_window = vote_window
        self.min_area = min_area
        self.vote_history = collections.deque(maxlen=vote_window)

        # HSV 阈值定义
        self.color_ranges = {
            "red": [
                (np.array([0, 100, 100]), np.array([10, 255, 255])),
                (np.array([170, 100, 100]), np.array([180, 255, 255])),
            ],
            "green": [
                (np.array([35, 100, 100]), np.array([85, 255, 255])),
            ],
            "yellow": [
                (np.array([20, 100, 100]), np.array([35, 255, 255])),
            ],
        }

    def detect(self, frame):
        """
        输入: BGR ndarray
        输出: {state: "red"/"green"/"yellow"/None, confidence: float}
        """
        hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
        color_areas = {}

        for color_name, ranges in self.color_ranges.items():
            mask = np.zeros(hsv.shape[:2], dtype=np.uint8)
            for lower, upper in ranges:
                mask |= cv2.inRange(hsv, lower, upper)

            # 形态学开运算去噪
            kernel = np.ones((5, 5), np.uint8)
            mask = cv2.morphologyEx(mask, cv2.MORPH_OPEN, kernel)

            # 找轮廓，过滤小面积
            contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
            total_area = sum(cv2.contourArea(c) for c in contours if cv2.contourArea(c) > self.min_area)
            color_areas[color_name] = total_area

        # 取面积最大的颜色
        if max(color_areas.values()) == 0:
            current_vote = None
        else:
            current_vote = max(color_areas, key=color_areas.get)

        self.vote_history.append(current_vote)

        # 多帧投票
        votes = [v for v in self.vote_history if v is not None]
        if not votes:
            return {"state": None, "confidence": 0.0}

        from collections import Counter
        vote_count = Counter(votes)
        winner, count = vote_count.most_common(1)[0]
        confidence = count / len(votes)

        return {"state": winner, "confidence": round(confidence, 2)}
```

```python
class ObstacleDetector:
    """
    障碍物识别 - 复用 YOLOv8 检测结果，零额外算力
    与 VehicleDetector 共享同一次推理，只是过滤不同类别
    """

    # COCO 类别中可作为"障碍物"的 ID
    OBSTACLE_CLASSES = {
        0: "person",       # 行人
        1: "bicycle",      # 自行车
        11: "stop_sign",   # 停止标志
        13: "bench",       # 长椅/路障近似
        25: "umbrella",    # 雨伞
    }

    def __init__(self, conf=0.4):
        """
        conf: 障碍物检测置信度阈值（比车辆低一些，避免漏检）
        """
        self.conf = conf
        self.class_names = self.OBSTACLE_CLASSES

    def filter_obstacles(self, all_detections):
        """
        从 YOLOv8 全部检测结果中过滤出障碍物
        输入: YOLOv8 全部检测结果 [{bbox, class_id, confidence}]
        输出: [{bbox, class, confidence, is_obstacle: True}]
        """
        obstacles = []
        for det in all_detections:
            cls_id = det.get("class_id")
            if cls_id in self.OBSTACLE_CLASSES and det["confidence"] >= self.conf:
                obstacles.append({
                    "bbox": det["bbox"],
                    "class": self.OBSTACLE_CLASSES[cls_id],
                    "confidence": det["confidence"],
                    "is_obstacle": True,
                })
        return obstacles

    def check_danger(self, obstacles, vehicles):
        """
        判断障碍物是否构成危险（与车辆距离过近）
        输入:
            obstacles: 障碍物列表
            vehicles: 车辆检测列表
        输出: 事件列表
        """
        events = []
        for obs in obstacles:
            if obs["class"] == "person":
                # 检查行人与每辆车的距离
                for vehicle in vehicles:
                    dist = self._bbox_distance(obs["bbox"], vehicle["bbox"])
                    if dist < 100:  # 像素距离阈值
                        events.append({
                            "type": "obstacle_warning",
                            "description": f"行人靠近车辆（距离{dist}px）",
                            "severity": "critical",
                            "bbox": obs["bbox"],
                        })
                        break
        return events

    @staticmethod
    def _bbox_distance(bbox1, bbox2):
        """计算两个 bbox 中心点之间的像素距离"""
        cx1 = (bbox1[0] + bbox1[2]) / 2
        cy1 = (bbox1[1] + bbox1[3]) / 2
        cx2 = (bbox2[0] + bbox2[2]) / 2
        cy2 = (bbox2[1] + bbox2[3]) / 2
        return ((cx1 - cx2) ** 2 + (cy1 - cy2) ** 2) ** 0.5
```

### ⑩ 车流密度热力图 — 累积叠加 + JET 伪彩映射

| 项 | 内容 |
|---|---|
| **算法** | 车辆位置累积叠加（`cv2.accumulateWeighted`）+ 高斯模糊 + JET 伪彩映射 + 原图叠加 |
| **输入** | ②的 track_id + bbox 中心点（每帧） |
| **输出** | 热力图图像（base64 编码）+ 热力区域坐标列表 |
| **额外耗时** | ~3ms/帧（CPU） |
| **用途** | 前端可视化展示车辆密度分布、拥堵热点区域 |

**核心原理：**

```
每帧处理流程：

  ① 维护一个与画面等大的"热度累积矩阵"（float32，初始全 0）
      │
  ② 每帧把当前所有车辆 bbox 中心点位置的热度 +1
      │
  ③ 热度矩阵做指数衰减（旧数据逐渐淡化）
      accumulateWeighted(heat_layer, heat_accum, alpha=0.05)
      │
  ④ 归一化到 0-255 → 高斯模糊（让热力图更平滑）
      │
  ⑤ applyColorMap(heat, COLORMAP_JET) → 伪彩色热力图
      │
  ⑥ addWeighted(原图, 0.6, 热力图, 0.4) → 叠加到原画面
      │
  ⑦ 编码为 base64 �给前端 WebSocket 推送
```

**核心代码逻辑：**

```python
import cv2
import numpy as np
import base64

class HeatmapGenerator:
    """
    车流密度热力图生成器
    - 实时累积车辆位置，生成密度热力图
    - 支持指数衰减（老数据淡出）
    - 输出叠加了热力图的画面 + 热力数据
    """

    def __init__(self, frame_size=(720, 1280), decay=0.05, blur_size=(25, 25)):
        """
        frame_size: (height, width) 画面尺寸
        decay: 衰减系数（0~1），越大衰减越快（0.05 = 大约 20 帧后淡化）
        blur_size: 高斯模糊核大小
        """
        self.heat_accum = np.zeros(frame_size, dtype=np.float32)
        self.decay = decay
        self.blur_size = blur_size

    def update(self, frame, tracked_objects):
        """
        输入:
            frame: BGR ndarray（当前帧）
            tracked_objects: [{track_id, bbox, ...}]
        输出:
            overlay_frame: 叠加了热力图的画面（BGR ndarray）
            heatmap_base64: 纯热力图的 base64 编码（给前端独立展示）
            hotspots: 热点区域列表 [{center, intensity}]
        """
        h, w = frame.shape[:2]

        # 1. 生成本帧热度层（当前车辆位置 +1）
        heat_layer = np.zeros((h, w), dtype=np.float32)
        for obj in tracked_objects:
            bbox = obj["bbox"]
            cx = int((bbox[0] + bbox[2]) / 2)
            cy = int((bbox[1] + bbox[3]) / 2)
            # 在中心点画一个填充圆（半径根据 bbox 大小自适应）
            radius = int(((bbox[2] - bbox[0]) + (bbox[3] - bbox[1])) / 6)
            cv2.circle(heat_layer, (cx, cy), max(radius, 10), 1.0, -1)

        # 2. 累积 + 指数衰减（旧热度逐渐淡出）
        cv2.accumulateWeighted(heat_layer, self.heat_accum, self.decay)

        # 3. 归一化到 0-255
        heat_norm = cv2.normalize(self.heat_accum, None, 0, 255, cv2.NORM_MINMAX)
        heat_norm = heat_norm.astype(np.uint8)

        # 4. 高斯模糊（让热力图更平滑自然）
        heat_blurred = cv2.GaussianBlur(heat_norm, self.blur_size, 0)

        # 5. JET 伪彩色映射
        heat_color = cv2.applyColorMap(heat_blurred, cv2.COLORMAP_JET)

        # 6. 叠加到原图
        overlay = cv2.addWeighted(frame, 0.6, heat_color, 0.4, 0)

        # 7. 提取热点区域（高热度连通域）
        hotspots = self._extract_hotspots(heat_blurred)

        # 8. 编码纯热力图为 base64（前端可独立展示）
        _, buffer = cv2.imencode(".jpg", heat_color, [cv2.IMWRITE_JPEG_QUALITY, 70])
        heatmap_b64 = base64.b64encode(buffer).decode("utf-8")

        return overlay, heatmap_b64, hotspots

    def _extract_hotspots(self, heat_blurred, threshold=128, min_area=500):
        """
        从热力图中提取热点区域
        输入: 灰度热力图
        输出: [{center: [cx, cy], intensity: 0.85, bbox: [x1,y1,x2,y2]}]
        """
        # 二值化
        _, binary = cv2.threshold(heat_blurred, threshold, 255, cv2.THRESH_BINARY)
        # 找连通域
        contours, _ = cv2.findContours(binary, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

        hotspots = []
        for c in contours:
            area = cv2.contourArea(c)
            if area < min_area:
                continue
            M = cv2.moments(c)
            if M["m00"] == 0:
                continue
            cx = int(M["m10"] / M["m00"])
            cy = int(M["m01"] / M["m00"])
            x, y, w, h = cv2.boundingRect(c)
            # 该区域的平均热度强度
            mask = np.zeros_like(heat_blurred)
            cv2.drawContours(mask, [c], -1, 255, -1)
            intensity = float(cv2.mean(heat_blurred, mask=mask)[0] / 255)

            hotspots.append({
                "center": [cx, cy],
                "intensity": round(intensity, 2),
                "bbox": [x, y, x + w, y + h],
            })

        # 按热度强度排序
        hotspots.sort(key=lambda x: x["intensity"], reverse=True)
        return hotspots[:5]  # 最多返回 5 个热点

    def reset(self):
        """重置热度累积矩阵（切换摄像头/重置时调用）"""
        self.heat_accum[:] = 0
```

| seaborn heatmap | Python 静态图 | 美观 | 非实时 | 不适合 |
| Kalman Filter 密度场 | 概率密度估计 | 理论更严谨 | 计算量大 | 过度设计 |

## 三、完整推理流水线

将上述 10 个模块组合为 pipeline，每帧调用一次：

```python
class InferencePipeline:
    """完整推理流水线"""

    def __init__(self, conf=0.5, iou=0.45, fps=15):
        # 初始化各模块
        self.detector = VehicleDetector(conf=conf, iou=iou)  # 同时检测车辆+障碍物
        self.obstacle_checker = ObstacleDetector(conf=0.4)
        self.tracker = VehicleTracker()
        self.flow_counter = FlowCounter()
        self.speed_estimator = SpeedEstimator(fps=fps)
        self.plate_recognizer = PlateRecognizer()
        self.congestion_detector = CongestionDetector(fps=fps)
        self.accident_detector = AccidentDetector(fps=fps)
        self.light_detector = TrafficLightDetector()
        self.heatmap_gen = HeatmapGenerator(frame_size=(720, 1280), decay=0.05)

        self.frame_id = 0

    def process(self, frame):
        """
        输入: BGR ndarray
        输出: InferenceResult JSON
        """
        import time as _time
        self.frame_id += 1

        # ① 车辆检测 + ⑨ 障碍物识别（一次推理，两个输出）
        vehicles, obstacles = self.detector.detect(frame)

        # ② 跟踪（仅跟踪车辆）
        tracked = self.tracker.track(frame)

        # ③ 车流量统计
        flow_stats = self.flow_counter.update(tracked)

        # ④ 车速估计
        speeds = self.speed_estimator.estimate(tracked)

        # ⑤ 车牌识别（仅对置信度高的车辆）
        detections = []
        for obj in tracked:
            tid = obj["track_id"]
            bbox = obj["bbox"]
            detection = {
                "track_id": tid,
                "bbox": bbox,
                "class": obj["class"],
                "confidence": obj["confidence"],
                "speed_kmh": speeds.get(tid),
                "plate": None,
            }

            # 只对置信度 > 0.7 的 car 类做车牌识别
            if obj["confidence"] > 0.7 and obj["class"] == "car":
                vehicle_img = frame[bbox[1]:bbox[3], bbox[0]:bbox[2]]
                if vehicle_img.size > 0:
                    plate = self.plate_recognizer.recognize(vehicle_img)
                    detection["plate"] = plate

            detections.append(detection)

        # ⑥ 拥堵检测
        congestion = self.congestion_detector.detect(
            flow_stats["current_count"], speeds
        )

        # ⑦ 事故检测
        events = self.accident_detector.detect(tracked, speeds)

        # ⑨ 障碍物危险判定（行人与车辆距离过近）
        obstacle_events = self.obstacle_checker.check_danger(obstacles, tracked)
        events.extend(obstacle_events)

        # ⑧ 红绿灯识别
        light = self.light_detector.detect(frame)

        # ⑩ 车流密度热力图
        overlay_frame, heatmap_b64, hotspots = self.heatmap_gen.update(frame, tracked)

        # 组装结果
        return {
            "frame_id": self.frame_id,
            "timestamp": _time.strftime("%Y-%m-%dT%H:%M:%S"),
            "image_annotated": None,  # 可改为 overlay_frame 的 base64
            "heatmap": heatmap_b64,   # 纯热力图 base64
            "hotspots": hotspots,     # 热点区域列表
            "detections": detections,
            "obstacles": obstacles,   # 障碍物列表
            "traffic_stats": {
                **flow_stats,
                "avg_speed": round(sum(speeds.values()) / len(speeds), 1) if speeds else 0,
                "congestion_level": congestion,
            },
            "events": events,
            "traffic_light": light if light["state"] else None,
        }
```

## 四、算法对比汇总表

| 功能 | 选用方案 | 备选方案 | 选择理由 |
|------|----------|----------|----------|
| 车辆检测 | YOLOv8-nano | Faster R-CNN / SSD512 | CPU 15FPS 实时，37+ mAP |
| 多目标跟踪 | ByteTrack | DeepSORT / SORT | 无需 ReID 模型，2ms/帧 |
| 车流量统计 | ROI 跨线计数 | 光流法 | 简单可靠，依赖跟踪 ID |
| 车速估计 | 像素位移标定法 | 光流估计 / 深度学习 | 无需训练，标定即可用 |
| 车牌识别 | HyperLPR3 | PaddleOCR / EasyPR | 车牌专用，95%+ 精度，5-10ms/张，内置透视矫正 |
| 拥堵检测 | 密度+速度双阈值 | 机器学习分类 | 可解释性强，参数可调 |
| 事故检测 | 规则法 | 深度学习异常检测 | 无需训练数据，规则可解释 |
| 红绿灯识别 | HSV 颜色阈值 | YOLO 检测 | 无需训练，<1ms/帧 |
| 障碍物识别 | YOLOv8 COCO 扩展类别 | 自定义训练障碍物模型 | 复用检测结果，零额外算力 |
| 车流密度热力图 | accumulateWeighted + JET 伪彩 | ECharts Heatmap / seaborn | 实时性好（3ms/帧），效果直观 |
